# TicketForge - Take‑Home Assignment

## Overview

TicketForge is a command-line tool for managing tickets. It implements a basic ticket‑management system
in Python. The goal is to allow users to create, list, and manage
tickets through a CLI interface.

## Features

- Create new tickets.
- List existing tickets
- Update ticket status or fields

## Repository Structure

    /config/             – configuration files
    /tickets_manager/    – core ticket‑management module
    main_cli.py          – main CLI entrypoint

## Getting Started

### Prerequisites

- Python 3.11

### Installation and Setup

    git clone https://github.com/Yahavba/TicketForge-Take-Home-Assignment.git
    cd TicketForge-Take-Home-Assignment
    activate python environment
    pip install -r requirements.txt

## Usage

Run the CLI to interact with the ticket system:

    python main_cli.py
